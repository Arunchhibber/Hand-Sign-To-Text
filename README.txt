RequirementsHardware* Windows PC or Laptop* Webcam (built-in or external)software* Python 3.7 or higher Ð to run the application.* Libraries/Packages: opencv-python = Captures video and processes imagestensorflow -Loads and runs the trained modelkeras - Loads the deep learning model and performs predictionspillow - Displays images in the GUInumpy - Array and numerical operationsinstallation / Setup* How to set up the project on a new machine:1. Install Python 3.7+2. Clone the project repository or place the folder in any location u wish u run it3. Install dependencies: Command to install them:pip install opencv-python tensorflow keras pillow numpy4. Ensure the trained model is in the project folder.5. Place button images in the img/ folder.Notes* Ensure all image files are in the img/ folder as used in the code.* The application saves recognized letters automatically after the gesture remains stable for 4 seconds.* The interface and button images can be customized without modifying the core detection code.
